# Frontend - Serviços de TI (AV2) • Versão aprimorada

- React + Vite
- Layout em tema escuro, responsivo
- Páginas:
  - Home institucional (história, vídeo, galeria, serviços, fundadores, contatos)
  - Login
  - Troca de senha
  - Cadastro de cliente
  - Carrinho de serviços de TI
  - Cadastro de serviços de TI

## Rodando

```bash
cd frontend
npm install
npm run dev
```

A aplicação estará em `http://localhost:5173`.
